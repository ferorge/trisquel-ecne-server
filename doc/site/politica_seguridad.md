# Política de seguridad

---------------------------------------------------------------------
##### _Aviso de seguridad importante_

---------------------------------------------------------------------

- Para la seguridad de los usuarios, este servidor Linux público está  
sujeto a estrictas políticas y medidas de seguridad diseñadas para  
proteger sus datos y garantizar el cumplimiento legal y legislativo.
- El alcance incluye todos los ordenadores vinculados y el acceso a  
Internet.
- A este servidor sólo acceden usuarios autorizados.
- Debe obtener el consentimiento explícito del administrador antes de  
intentar conectarse a este servidor.
- Cualquier uso no autorizado de este sistema está prohibido y puede  
considerarse un acto ilegal.
- Cualquier evidencia de intrusión, acto delictivo, acceso no  
autorizado o intento de eludir las medidas de seguridad puede ser  
penalizado.

---------------------------------------------------------------------
__Advertencia: al iniciar sesión en este sistema, usted reconoce y__  
__acepta el seguimiento y registro de actividades.__

---------------------------------------------------------------------

El seguimiento y registro de actividades es continuo y es realiado  
únicamente con el objetivo de auditar y garantizar la seguridad de  
nuestros usuarios y sistemas.  
No se utiliza para espionaje, seguimiento ni otra finalidad distinta  
a la mencionada.

---------------------------------------------------------------------
##### _Privacidad y protección de datos_

---------------------------------------------------------------------
- Los usuarios y administradores de este servidor se comprometen a  
hacer cumplir la política de seguridad y las leyes y normativas  
aplicables, protegiendo la privacidad y garantizando la  
confidencialidad de los datos de todos los usuarios.
- Los datos son privados, su uso está restringido y cada usuario es  
propietario de los suyos.
- Los datos se almacenan de forma segura y se prohíbe el acceso no  
autorizado.
- Sin embargo, no podemos garantizar la seguridad de los datos.  
transmitidos a través de Internet y le recomendamos que tenga cuidado  
al transmitir información confidencial.

---------------------------------------------------------------------
##### _Buenas prácticas de seguridad_

---------------------------------------------------------------------

1. Utilice contraseñas seguras y cámbielas periódicamente.
2. No compartas tus credenciales de acceso con nadie.
3. Desconecta o bloquea tu sesión cuando no la estés usando.
4. Evite acceder a enlaces sospechosos.
5. Verificar la autenticidad de los sitios web antes de ingresar  
información confidencial.
6. Almacene datos confidenciales de forma segura.
7. Eliminar archivos temporales.

---------------------------------------------------------------------
__Gracias por aceptar estos términos y condiciones y por tu__  
__cooperación para mantener la seguridad de todos nosotros.__
