# Código de conducta

#### __Objetivo__

1. Establecer un marco de referencia para las decisiones y acciones de todas  
las personas usuarias y administradoras.
2. Establecer el conjunto de principios, valores y reglas para definir las  
expectativas de comportamiento.  
3. Promover la integridad, la responsabilidad, la equidad, la transparencia y  
la buena voluntad de la comunidad.
