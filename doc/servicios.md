# Servicios disponibles

---------------------------------------------------------------------
##### _Servicios_

---------------------------------------------------------------------
Este servidor ofrece una variedad de servicios y aplicaciones para  
usuarios finales que necesitan trabajar con bases de datos,  
aplicaciones web, archivos y comunicaciones seguras en red.  
  
    
- [__Apache__][apache2]: es un popular servidor web utilizado para  
alojar y servir sitios y aplicaciones web escritos en HTML, CSS,  
JavaScript y otros lenguajes de programación web. 
- [__MariaDB__][mariaDB]: es una base de datos relacional compatible  
con MySQL que se puede utilizar para almacenar y gestionar grandes  
cantidades de datos estructurados.
- [__Python 3__][python3]: es un popular lenguaje de programación  
interpretado que se puede usar para crear scripts, herramientas y  
aplicaciones web complejas.
- [__Finger__][finger]: es un programa de cliente/servidor que permite  
obtener información sobre los usuarios de un sistema remoto, como su  
nombre real, shell y último inicio de sesión. 
- [__Tor__][tor]: es una red anónima y descentralizada que permite a  
los usuarios navegar por Internet de forma anónima y evitar la  
vigilancia y censura en línea. 
- [__TTYD__][ttyd]: es un multiplexador terminal gráfico basado en  
websockets que permite abrir varias consolas remotas dentro de un solo  
navegador web.
- [__Gopher__][gopher]: es un protocolo de red simple que precedió al  
World Wide Web. Se utiliza principalmente para publicar documentos  
textuales planos y recursos binarios simples en un formato similar a  
un menú jerárquico.
- [__Bind__][bind9]: es un servidor de nombres de dominio (DNS) que  
convierte nombres de host legibles por humanos en direcciones IP  
numéricas.
- [__FTP__][ftp]: es un protocolo estándar para transferencia de  
archivos entre dos hosts a través de TCP/IP. Permite subir y bajar  
archivos desde un servidor remoto, ya sea de manera interactiva o  
automatizada.

- __Squid__: es un proxy HTTP/HTTPS y FTP que mejora el rendimiento y controla el acceso a Internet. Puede funcionar como caché forward o reverse, reduciendo así el tiempo de descarga de contenido repetitivo.


- __Gemini__: es un protocolo ligero y minimalista similar a Gopher pero diseñado especialmente para navegación en texto sin formatos complicados ni multimedia.

- __IRC__: es un protocolo de mensajería instantánea que permite a los usuarios chatear en canales públicos o privados.
